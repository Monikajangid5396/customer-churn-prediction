# 📊 Customer Churn Prediction System

## 📌 Project Overview

This project predicts whether a telecom customer is likely to churn (leave the company) or stay using Machine Learning techniques.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Machine Learning Models
- Streamlit Web Application

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit
- Joblib
